**Синдром Барттера, тип 4B (неонатальный), с нейросенсорной глухотой** (OMIM: [613090](https://omim.org/entry/613090))


# 1. Описание

Это редкое наследственное заболевание, относящееся к тубулопатиям (нарушениям работы почечных канальцев). Болезнь проявляется уже во внутриутробном периоде или сразу после рождения и характеризуется нарушением транспорта ионов в почках.

# 2. Ассоциированные гены

Для OMIM 613090 характерна дигенная наследственность (нужны мутации в двух генах одновременно):

1) CLCNKA
Кодирует хлоридный канал (ClC-Ka)
Участвует в реабсорбции солей в почках и работе внутреннего уха
Связан с данным фенотипом напрямую
2) CLCNKB
Похожий хлоридный канал (ClC-Kb)
Также участвует в транспорте ионов в почках
Совместные мутации с CLCNKA -> тип 4B

# 3. Последовательности этих генов

Файлы в папке `fasta_files`:
 - `CLCNKA_human.fasta` - CLCNKA для человека
 - `CLCNKA_mouse.fasta` - CLCNKA для домашней мыши
 - `CLCNKB_human.fasta` - CLCNKB для человека
 - `CLCNKB_human.fasta` - CLCNKB для домашней мыши

# 4. Парное выравнивание

Файлы в папке `aln`:
 - `clcnka_needle.txt` - выравниание CLCNKA человека и мыши с помощью needle 
 - `clcnka_water.txt` - выравниание CLCNKA человека и мыши с помощью water
 - `clcnkb_needle.txt` - выравниание CLCNKB человека и мыши с помощью needle
 - `clcnkb_needle.txt` - выравниание CLCNKB человека и мыши с помощью water
